SerialtoSevenSegment

Simple Arduino code for interfacing with a seven segment display and the serial port.
It is MIT Licensed, so do with it what you want.
Here is the pinout diagram:



Grab the code:


git clone https://github.com/opnitro/SerialtoSevenSegment

How to use


Transfer the code to the arduino, and build the circuit. Once that is done, go to Tools -> Sereal Moniter (command-shift-M). Enter a single digit number and press enter, it should appear on the display. Sending H, with bring up a 9 with a dot next to it, representing more than 9. Sending O will turn off the display.


An example of sending in python:
s = serial.Serial('/dev/tty.', 9600)
s.write('1')
Enjoy. :D
